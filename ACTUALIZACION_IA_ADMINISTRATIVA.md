## 💬 Ejemplos de Uso

### Crear Usuarios
```
Usuario: "Crea un nuevo usuario llamado Juan Pérez con email juan@agroverde.com"
IA: "Perfecto, voy a crear el usuario Juan Pérez. Necesito confirmar algunos datos..."
```

### Registrar Operaciones
```
Usuario: "Registra una pesada de 500kg de café del cliente ABC"
IA: "Entendido, voy a registrar la pesada. Necesito la fecha y algunos detalles adicionales..."
```

### Gestionar Inventario
```
Usuario: "Agrega 100 unidades de Café Premium al inventario"
IA: "Voy a agregar el producto al inventario. ¿Cuál es el precio unitario?"
```

### Consultar Información
```
Usuario: "¿Cuántas facturas pendientes tenemos?"
IA: "Déjame consultar las facturas pendientes en el sistema..."
```

## 🚀 Cómo Usar

1. **Abre el asistente de IA** (botón flotante en la esquina inferior derecha)

3. **La IA te guiará** paso a paso para completar la operación
4. **Confirma los datos** cuando sea necesario
5. **¡Listo!** La operación se ejecuta
